# 13.3_tdd_Geometrie
Hier sollen die ersten Schritte mit TDD gemacht werden.

Klonen Sie das Repository und implementieren Sie, bis alle Testfälle erfolgreich sind.

Den Testfallexplorer können Sie in der Menüleiste unter dem Reiter `Test` finden.

Bei Problemen mit der dotNET-SDK-Version muss für beide Projekte (Rechtsklick auf die Projektmappe) die Zielversion von dotNET auf ein installiertes SDK geändert werden.
